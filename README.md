# Slope-Intercept Explorer

An interactive browser-based tool for exploring the relationship between coordinates, slope, y-intercept, and the equation **y = mx + b**.

## Features

- **Interactive graph** with a draggable line — grab the control points to change slope and intercept
- **Live equation display** — editable m and b values update everything in real time
- **Sliders** for slope and y-intercept
- **Editable data table** — change point values and see the best-fit line update via linear regression
- **Bidirectional sync** — all controls stay perfectly in sync

## Usage

Open `index.html` in any modern browser, or visit the GitHub Pages deployment.

No build step or dependencies required.
